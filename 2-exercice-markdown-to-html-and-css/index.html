Titre - Headers

Pour convertir un texte en titre, il suffit d'ajouter un ```#``` avant.


Exemple:

Ceci est un <h1>

Ceci est un <h2>

Ceci est un <h3>

Ceci est un <h4>

Ceci est un <h5>
Ceci est un <h6>

Veuillez à bien laisser un espace entre le # et le texte !

Les liens

Liens internes/ancres

Taper la commande :
<a href="#ancre">Return to Top</a>
Ce qui va donner:
Return to Top

Liens externes

Taper la commande:
<a href="http://www.google.fr target= "_blank"</a>google.fr
Ce qui va donner:
google.fr"

Les citations

Taper le code suivant permettra de créer un bloc de citation en respectant les règles d'usage :
<blockquote>La pierre et le rocher coulent dans l'eau de la même façon
<small>- Auteur inconnu</small>, extrait du film <cite>Old Boy</cite> </blockquote>
Cela va donner :
La pierre et le rocher coulent dans l'eau de la même façon
- Auteur inconnu, extrait du film Old Boy
